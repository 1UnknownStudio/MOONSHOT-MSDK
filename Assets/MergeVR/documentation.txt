1. Extension Base

Merge Input

MergeVR Headset Left / Right Capacitive Touch Buttons

The production version of the MergeVR headset is equipped two indirect touch buttons. The state of these buttons can be determined with the following calls.

Merge.MergeInputCardboard.GetInput(N) //Where N is 0 for right button and 1 for left button
Merge.MergeInputCardboard.GetInputDown(N)
Merge.MergeInputCardboard.GetInputUp(N)

Merge.MergeInputCardboard.GetSingleInput() // returns true on any button touch
Merge.MergeInputCardboard.GetSingleInputDown()
Merge.MergeInputCardboard.GetSingleInputUp()
Merge.MergeInputCardboard.GetDoubleInput() // returns true if both left and right buttons are pressed simultaneously
Merge.MergeInputCardboard.GetDoubleInputUp() 
Merge.MergeInputCardboard.GetDoubleInputDown() 



2. Example Scenes


DemoLeftInput

A demo of how to add MergeVR headset left trigger detection to your existing cardboard Unity project as simply as possible.

- For this demo set the 'Tap is Trigger' on the CardboardMain object to false

- Run and test on iOS or Cardboard - on a standard Cardboard device only one square will highlight at button press - with MergeVR headset you can touch left, touch right, and double touch (both buttons highlighted).

- Example C# code is in script MergeDemo
